User Need

I want to get back directly to the starting point (the home page of a site or relevant main page of a set of web pages) when I get lost/disoriented,

Related User Story: Placeholder.

What to Do

Make sure starting points are:

  • easy to get to,
  • easy to orient,
  • easy to find on each screen.

For example: Make the starting point more visible and identifiable by using a predictable location, larger sizes, strong color contrast, and appealing icons for the home page logo or links.

Note: A site can have a different starting point for subsites and sections of the site. For example, each facility of a university may have its own home page. In such a case the home page of both the site and the subsite that the user is on should be easy to get to. Placeholder-Todo add clarification of main page/landing page, and structure.

How it Helps

Users often get lost and disoriented when using apps and websites. This is often aggravated by stress, mental health, and/or cognitive overload. Providing an easily identifiable starting point helps users know, they can always restart if they become disoriented or lost.

Making starting points more visible and recognizable provides a more efficient way for people with cognitive disabilities to reorient themselves to complete their intended tasks.

For example, a user with a cognitive disability navigates a wrong path during a search attempt or gets lost while exploring a website. If a home page link and logo are easily identifiable, the user can directly go to the home page.It assures them that they can restart anytime if lost. They don't need to press the back button multiple times. This helps them restart their browsing task quickly. It prevents the negative impact on their efficiency that may be caused by reading every page on the way back to the home page.

More Details

Some large websites use templates are for subsites or different sets of pages.They often have more than one starting point. Examples of sites that may have more than one starting point includes:

  • An online retail shop may have a site with a home page and a department main page with a specific color-coded template.
  • For a banking site,the bank home page and an account main page.
